# Data Folder

This folder contains datasets used in the **Air Quality Prediction and Alert System** project.

---

## Datasets Included

###  Raw Dataset
- **city_day.csv:** raw_dataset.csv  
- **Description:**  
  This dataset contains the original air quality data collected from sources without any preprocessing.  
- **Features may include:**  
  - PM2.5  
  - PM10  
  - NO2  
  - SO2  
  - CO  
  - O3  
  - Date/Time  
  - City/Location  

 This dataset is used as the **initial input for data cleaning and preprocessing**.

---

### Preprocessed Dataset
- **aqi_data.csv:** preprocessed_data.csv  
- **Description:**  
  This dataset is the cleaned and processed version of the raw dataset.  
- **Preprocessing steps include:**  
  - Handling missing values  
  - Removing duplicates  
  - Feature selection    

This dataset is used for **training the machine learning model**

##  Purpose
These datasets are used to:
- Train AQI prediction models  
- Analyze air quality trends  
- Generate alerts and health recommendations
